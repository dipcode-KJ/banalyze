Service Guide

B.LEAGUE統計Webサービス
機能説明書

NINES DATA ANALYZEは、Bリーグの試合結果を収集し、勝率・得失点差・ホーム/アウェイ傾向・直近フォームなどをシーズン別に可視化するデータ分析サービスです。

Project NINES DATA ANALYZE
Target analyze.kjnine.com
Data B1 / B2・過去シーズン対応
Generated 2026-05-25

1. サービス全体でできること

シーズン切替

2025-26、2024-25、2023-24を切り替えて、同じ形式でチーム成績・試合結果・推移を確認できます。

ディビジョン切替

ALL、B1、B2を切り替え、全体・B1のみ・B2のみのランキングや分析を比較できます。

DB/API連携

MySQLに保存した試合データと集計データをPHP APIから取得し、画面に反映します。APIが使えない場合はモックデータへフォールバックします。

グラフ可視化

Chart.jsで勝率、得失点差、スコア比較、攻撃/守備ポジション、フォームなどを視覚的に確認できます。

2. 共通ナビゲーション・操作

ヘッダー ロゴ、Overview、Teams、Analytics、Trends、Resultsへの導線があります。各ページ間を直接移動できます。
シーズン選択 各画面で対象シーズンを切り替えます。切替後、勝率・得失点差・試合結果・ランキングが選択シーズン基準で再計算されます。
ディビジョン選択 ALL、B1、B2を選択できます。B1とB2を混ぜた全体傾向も、カテゴリ別の傾向も見られます。
フッター サービス情報、ページリンク、データ利用上の注意を掲載します。公開サービスとしての信頼感を補強する領域です。

3. Overviewページ

URL: /

サービスの入口となるダッシュボードです。選択したシーズン・ディビジョンの主要指標、ランキング、最新試合をまとめて確認できます。

メインビジュアル 現在の選択条件における勝率トップチーム、勝率、得失点差、上位チームの簡易バーを表示します。
主要指標 最高勝率、対象試合数、平均得点、データソース（DB/APIまたはモック）をカード形式で表示します。
チーム勝率 上位チームの勝率を棒グラフで比較します。どのチームが勝率面で抜けているかを素早く把握できます。
得失点差推移 上位チームの得失点差を折れ線グラフで表示します。勝率だけでは見えない得点力・守備力の差を見られます。
勝率トップチームの過去推移 現在トップのチームが過去シーズンでどの程度の勝率だったかを確認できます。単年の強さか、継続的な強さかを判断できます。
チーム別成績 チーム名、地区、勝敗、勝率、HOME、AWAY、得失点差、直近5試合を一覧で確認できます。
最新試合結果 直近の試合日、対戦カード、スコア、会場をカードで確認できます。

4. Teamsページ

URL: /teams

チームごとの強さを、ホーム/アウェイ成績と得失点差から分解して見るページです。

ホーム・アウェイ勝率 各チームのHOME勝率とAWAY勝率を並べて比較します。ホームで強いチーム、アウェイでも安定するチームを見分けられます。
得失点差ランキング 得点と失点の差が大きい順にチームを表示します。勝敗だけでなく、試合内容の優位性を確認できます。
詳細チームスタッツ 勝率、勝敗、平均得点、平均失点、得失点差、HOME成績、AWAY成績を一覧で確認できます。

5. Analyticsページ

URL: /analytics

チームの強さを多角的な指標で読むための分析ページです。単純な勝率ランキングより一段深く、攻撃・守備・安定度・直近状態を比較できます。

総合スコア上位カード 総合指標が高い上位チームをカード形式で表示します。攻撃、守備、直近フォームをまとめて確認できます。
攻撃・守備ポジション 平均得点と平均失点を散布図で表示します。攻撃型、守備型、バランス型のチーム傾向を直感的に見られます。
ホーム依存度 HOME勝率とAWAY勝率の差を表示します。ホームで強いが外で落ちるチーム、環境に左右されにくいチームを判断できます。
直近フォームと安定度 直近5試合の勝利割合と、得失点差・ホーム依存度から算出した安定度を比較します。
多角指標マトリクス 総合、攻撃、守備、勝率、ホーム依存、直近フォーム、安定度をチーム別に一覧化します。

Analyticsページの使いどころ: 勝率が近いチーム同士を比較するとき、攻撃力で勝っているのか、守備で支えているのか、直近の勢いがあるのかを分解できます。

6. Trendsページ

URL: /trends

過去シーズンをまたいだ変化を見るページです。選択条件における上位チームの推移や、シーズン内の得失点差傾向を確認できます。

上位チームの勝率推移 選択シーズンで上位のチームについて、過去シーズンの勝率推移を折れ線で表示します。継続的に強いチームと急伸したチームを見分けられます。
選択シーズンの得失点差 選択中のシーズン・ディビジョンにおける得失点差を棒グラフで比較します。
上位チーム概要 上位4チームの勝率と得失点差をコンパクトに表示します。ページ下部で要点だけ確認できます。

7. Resultsページ

URL: /results

試合単位の結果を確認するページです。最新試合、スコア比較、合計得点の高い試合を見られます。

試合別スコア比較 直近試合のHOME得点とAWAY得点を棒グラフで比較します。接戦、大差、ホーム優勢などを視覚的に確認できます。
合計得点ランキング HOME得点とAWAY得点の合計が高い試合をランキング表示します。ハイスコアゲームを探せます。
試合結果カード 試合日、ホームチーム、アウェイチーム、スコア、会場をカード形式で一覧表示します。

8. データ取得・管理でできること

試合データ投入 cron/import-games.php がB.LEAGUEのスケジュール/結果データを取得し、MySQLに保存します。
過去シーズン投入 php cron/import-games.php 2024-25 backfill のようにシーズンを指定して、過去シーズンをまとめて取り込めます。
集計再生成 cron/rebuild-records.php により、既存の試合データから勝敗・得失点・HOME/AWAY成績を再集計できます。
ヘルスチェック /api/health.php でDB接続状態、teams/games/records件数、最新試合日を確認できます。
公開環境 Xserver上で静的フロント、PHP API、MySQL、Cronを組み合わせて運用できます。

9. 想定される利用シーン

  • シーズンごとの上位チーム、勝率、得失点差をざっくり把握する。
  • B1とB2を分けて、カテゴリごとの強さや試合傾向を確認する。
  • 勝率だけでは見えない、平均得点・平均失点・ホーム依存度を比較する。
  • 直近フォームを見て、今勢いがあるチームを把握する。
  • 過去シーズンとの比較で、継続的に強いチームや急伸したチームを見つける。
  • 試合結果カードから、直近の対戦結果やハイスコアゲームを確認する。
